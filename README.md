# TrevelGuide

## installation

1. clone repository
~~~
git clone https://github.com/thisisfunt/TrevelGuide.git
~~~
2 run frontend
~~~
cd TrevelGuide/frontend
~~~
~~~
npm start
~~~
3 run backend
~~~
cd backend
~~~
~~~
pip install -r requirements.txt
~~~
~~~
cd backend
~~~
add info about database in settings.py
~~~
python manage.py runserver
~~~
